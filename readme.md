# cookiecutter-template

My meta cookiecutter template

## Requirements
Install `cookiecutter` command line: `pip install cookiecutter`    

## Usage
Generate a new Cookiecutter template layout: `cookiecutter gh:denislaliberte/cookiecutter-template`
